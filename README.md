## MU for mobile "穆"

##### 包含:

    - core          核心
    - utility       常用工具方法集合
    - detect        平台检测
    - ajax & cache  请求相关
    - touch event   基础touch事件
    - fn            拓展$.fn
    - require       简易require

##### 组件

    - dialog        弹窗
    - slidepage     页面转场
    - slider        轮播
    - calendar      日历


##### how to use

    - git clone to your local enviroment
    - npm install
    - grunt server
    - do what you want
